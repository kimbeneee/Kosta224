- 요구사항
회원관리
비회원상태에서 아이디에 의한 회원검색이 가능하다
아이디에 대한 회원이 존재한다면 검색된 회원 결과는 아이디, 이름 , 주소, 생년월일, 가입일시가 제공된다
생년월일은 ( 2021.10.21 ) 연.월.일 형식으로 제공되고
가입일시는 연.월.일 시.분.초 로 제공된다

로그인 기능이 제공된다
로그인 성공하면 메인화면에 로그인폼 대신 회원이름이 제공되고 로그아웃 링크가 제공된다
로그아웃을 클릭하면 다시 로그인폼에 제공된다

로그인 상태일 때는 주소로 회원리스트 검색이 가능하다
회원리스트는 테이블 형식으로 아이디 이름 생년월일 을 제공한다

로그인 상태일때는 회원정보 수정이 가능하다
회원정보 수정은 패스워드, 이름, 주소, 생년월일 을 수정할 수 있다
수정 후 수정된 회원정보를 결과화면에서 제공한다

회원가입시에 아이디 패스워드 이름 주소 생년월일을 입력받아 가입하고 등록일시는 시스템에서 처리한다
아이디 중복확인 과정을 거쳐야만 회원가입 되도록 한다
